# Premmerce Test Task
Used technologies:
  - HTML5
  - SCSS/CSS
  - Gulp
## Before project run
Clone repository
Be sure to have installed gulp and gulp-cli globally
```sh
$ npm i gulp gulp-cli -g
```
Install dependencies
```sh
$ npm i
```
## Build project
```sh
$ gulp build
```
or
```sh
$ npm run build
```
## Run project
Go to dist folder
```sh
$ cd dist
```
Open "index.html" in any browser
